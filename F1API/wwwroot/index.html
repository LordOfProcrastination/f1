Introduction

Welcome to the documentation of my F1API
With this API you can do full CRUD on the driverDB containing information on all participants of the F1 Whacky Races event

http://localhost:5139/api

Get All Drivers

    Possible errors:
  • 500 - Internal Server Error

You can access a list of all drivers by using the /driver endpoint

GET http://localhost:5139/api/driver

{"id":1,"name":"Lewis Hamilton","age":38,"nationality":"British","team":"Mercedes", "vehicle":"W14","isProfessional":true,"image":"lewis-hamilton.png"}

Get A Single Driver

    Possible errors:
  • 404 - Not Found
  • 500 - Internal Server Error

You can get a single driver using id as a parameter /driver/2

GET http://localhost:5139/api/driver/2

{"id":2,"name":"George Russell","age":25,"nationality":"British","team":"Mercedes", "vehicle":"W14","isProfessional":true,"image":"george-russel.png"}

Get Driver By Professionality

    Possible errors:
  • 404 - Not Found
  • 500 - Internal Server Error

You can get either professional or amateur drivers using getprofessional as a parameter getprofessional as a parameter /driver/getprofessional/false

GET http://localhost:5139/api/driver/getprofessional/false

{"id":21,"name":"Mysil Bergsprekken","age":30,"nationality":"Norwegian","team":"Red-Bull","vehicle":"Boomerang Rapido","isProfessional":false,"image":"mysil-bergsprekken.png"}

Post A Driver

    Possible errors:
  • 404 - Not Found
  • 500 - Internal Server Error

You can post a driver in the database as long as you pass all required values /driver/3

POST http://localhost:5139/api/driver/3

{"id":3,"name":"Max Verstappen","age":26,"nationality":"Dutch","team":"Red-Bull", "vehicle":"RB19","isProfessional":true,"image":"max-verstappen.png"}

Edit A Driver

    Possible errors:
  • 500 - Internal Server Error

You can edit a driver in the database by using id /driver/4

PUT http://localhost:5139/api/driver/4

{"id":4,"name":"Sergio Perez","age":33,"nationality":"Mexican","team":"Red-Bull", "vehicle":"RB19","isProfessional":true,"image":"sergio-perez.png"}

Delete A Driver

    Possible errors:
  • 404 - Not Found
  • 500 - Internal Server Error

You can delete a driver from the database by using id as a parameter /driver/5

DELETE http://localhost:5139/api/driver/5

{"id":5,"name":"Charles Leclerc","age":26,"nationality":"Monacan","team":"Ferrari", "vehicle":"SF-23","isProfessional":true,"image":"charles-leclerc.png"}
